javascript值

常见的js数据类型（值）:String Number Null Object Function

js值的分类：

  1. 原始值
  2. 引用值

原始值（基本类型、原始类型，5种）：

​ Undefined、Null、Boolean、Number、String

  1. 存储在栈中(stack)，直接存储在变量访问的内存中。
  2. 除String类型外，有固定大小。Number的固定大小是9007199254740992（2的53次方）

引用值（引用类型，对象Object）：

  1. 存储在堆中(heap)，变量中存储一个指针(point)，指向内存处。
  2. 没有固定大小

js三大对象

对象是带有属性和方法的特殊数据类型。

  1. 本地对象（也可叫原生对象，内部对象。就是上面的引用类型，需要实例化）

    实例化就是需要new，包含：Object、Array、Function、String等等。。。

  2. 内置对象（无需实例化）

    内置对象是本地对象的子集，无需new

    包含Math、Global（global没用），es5中添加了JSON（一共仨）

  3. 宿主对象（嵌入到哪，哪就是宿主）

    嵌入到网页中，浏览器提供的就是宿主对象

    DOM对象：文档标准，操作文档（Document。。）

    BOM对象：浏览器功能（window、cookies。。）

如果按照是否是函数来分类对象，对象分为两种，函数对象与普通对象！
