json格式

# json

JavaScript Object Notation javascript对象表示法

通用的，描述对象的文本格式（独立于计算机语言与平台）

javascript用JSON描述对象（json格式与创建javascript对象相同）

json语法规则：

  • 数据以键值对形式存在
  • 键与值以:（冒号）分隔，键值对以,（逗号）分隔
  • 字符串需要用双引号 !!!
  • 花括号表示对象{}
  • 方括号表示数组[]

json值可以为：

  • 字符串（双引号）
  • 数字
  • Boolean( true,false)
  • null
  • 对象 {} //也不能有方法
  • 数组 []
var obj=[{"name":"dongfang","xing":"zhao"},{"age":"18"},[{"战斗力":"5"}]]
